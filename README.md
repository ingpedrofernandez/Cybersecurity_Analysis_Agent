# Cybersecurity_Analysis Agent

## A Code Analyzer via Semgrep MCP server for Azure and GCP

The Cybersecurity_Analysis Agent is an AI-powered web application that analyzes Python code for security vulnerabilities. It combines:

OpenAI's latest models for intelligent code analysis
Semgrep for static security scanning
A React/Next.js frontend
A FastAPI backend
Docker containerization
Cloud deployment with Terraform
